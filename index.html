What is DOM DOM defined a standard for accessing documents The W3C Document object model(DOM) is platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document. HTML dom is a standard object model and programing interface for HTML. It defines: The HTML element as objects The properties of all HTML The methods to access all HTML The events for alll HTML elements The html dom is standard for how to get, change, add, delete HTML elements.
